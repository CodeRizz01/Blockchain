# Blockchain
Blockchain is an immutable, sequential chain records called Blocks. They can contain transactions, files or any data you like. 
But the most important part is they are "chained" together using "hashes"

Hashes: A hash is a unique digital fingerprint generated by processing inpit like transaction records through a mathematical algorithm known as "Blockchain Hash Function". It compress any amount of data into a fixed length of string characters, securing network data and ensuring tamper-proof transaction
